# Code Of Conduct

This project aims to be welcoming, respectful, and constructive.

## Expected Behavior

- Be kind and professional.
- Give feedback that is specific and helpful.
- Assume good intent and ask clarifying questions when needed.
- Focus on improving the project rather than winning arguments.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or sustained disruption
- Publishing private or security-sensitive information without permission
- Dismissing contributors in a way that makes participation unsafe or hostile

## Maintainer Responsibilities

Maintainers may edit, remove, or reject comments, issues, pull requests, or other contributions that do not support a healthy project environment.

## Reporting

If you experience or witness harmful behavior, contact the maintainers through a private channel when possible. If a private channel is not available, use GitHub moderation and reporting tools as appropriate.
